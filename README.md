## 项目介绍
### 1. 项目概述
本项目旨在使用OpenCV计算机视觉库和Qt框架，开发一个功能完整的桌面面部识别应用程序。该系统能够从实时摄像头视频流或静态图片中检测人脸，提取其生物特征，并与一个已知人员数据库进行比对，从而实现身份识别。项目将重点集成先进的人脸检测与识别算法，并通过一个友好的图形用户界面进行人员数据库管理、实时识别展示和结果记录，旨在构建一个集数据采集、模型训练（或加载）、实时识别于一体的端到端人脸识别解决方案。

### 2. 核心功能与技术实现
#### 2.1 实时视频与图像处理（可选只通过图片识别）
- 功能描述：
    1. 摄像头视频流捕获：
        - 能够自动检测并打开系统连接的摄像头。
        - 在一个界面组件中实时、流畅地显示摄像头捕获的视频画面。
    2. 静态图片导入：允许用户从本地文件系统中加载单张图片进行人脸识别分析。
    3. 视频流控制：提供“开始/停止”按钮来控制视频流的捕获与处理。
- (关联Qt技术：QLabel (用于显示视频帧), QTimer (驱动视频帧更新), QtMultimedia (备选方案), QFileDialog)
- (关联OpenCV技术：cv::VideoCapture, cv::Mat, cv::cvtColor)

#### 2.2 人脸检测与特征提取
- 功能描述：
    1. 高精度人脸检测：
        - 在每一帧视频或加载的图片中，能够准确地检测出所有存在的人脸位置。
        - 在视频流中，用矩形框实时地、稳定地框出检测到的人脸。
    2. 特征向量提取 (Embedding)：
        - 对于每一个检测到的人脸，系统会使用一个预训练的深度学习人脸识别模型来提取其关键的、唯一的生物特征，并将其编码成一个高维的特征向量（例如，一个128维的浮点数数组）。这个向量是后续身份比对的核心依据。
- (关联OpenCV技术：cv::CascadeClassifier (Haar/LBP, 传统方法), cv::dnn::Net (加载深度学习模型如MTCNN, RetinaFace), cv::dnn::blobFromImage)
- (关联AI模型：FaceNet, ArcFace, DeepFace等预训练模型)

#### 2.3 人员数据库管理与“人脸注册”
- 功能描述：
    1. 人员信息管理界面：提供一个后台管理界面，用于增、删、改、查已知人员的信息（如ID、姓名、部门等）。
    2. 人脸注册（录入）：
        - 管理员可以通过摄像头为新人员拍摄多张不同角度、不同表情的人脸照片，或者从本地导入照片。
        - 系统会自动从这些照片中检测人脸，提取特征向量，并将这些向量与人员信息一起存入数据库。
        - 可以对一个人的多个特征向量求平均值，以获得更鲁棒的中心特征。
- (关联Qt技术：QTableView, QDialog, Model/View架构, QtSql (用于数据库))
- (关联数据存储：SQLite, 或者自定义的二进制/JSON文件)

#### 2.4 实时身份识别与结果展示
- 功能描述：
    1. 实时比对：在视频流中，系统会实时地将检测到的每张人脸的特征向量，与数据库中所有已知人员的特征向量进行相似度计算。
    2. 身份判定：
        - 如果计算出的最高相似度超过一个预设的阈值，系统就判定该人脸为数据库中的对应人员。
        - 如果最高相似度低于阈值，则判定为“陌生人”。
    3. 结果可视化：
        - 在视频画面的实时人脸框旁边，直接显示识别出的姓名或“陌生人”字样，并可以显示相似度得分。
        - 提供一个识别日志列表，按时间顺序记录每一次成功的识别事件（时间、姓名、抓拍快照）。
- (关联Qt技术：在QPixmap上绘制文本和矩形, QListView/QTextBrowser (for logs))

### 3. 算法与数据结构深度优化（可选、提升）
- 人脸识别性能优化：
    - 相似度计算算法：使用余弦相似度（Cosine Similarity）或欧氏距离（Euclidean Distance）来度量两个特征向量的相似程度。余弦相似度在处理高维向量时通常表现更佳。
    - 快速近邻搜索：当人员数据库非常大时（如成千上万人），线性地与库中每个人比对会非常慢。可以引入近似最近邻（ANN）算法和库（如FAISS, Annoy），通过构建KD树、LSH（局部敏感哈希）等数据结构，将O(N)的搜索复杂度大幅降低到近似O(log N)，实现大规模人员的毫秒级识别。
- 人脸跟踪与稳定性：
    - 目标跟踪算法：为了使视频中的人脸框更稳定、不闪烁，可以在人脸检测的基础上，引入简单的目标跟踪算法（如卡尔曼滤波器或基于相关滤波的跟踪器如KCF）。在检测失败的少数几帧里，由跟踪器来预测人脸的位置。
- 活体检测（防照片攻击）：
    - 算法：可以额外集成一个活体检测模型，通过分析人眼的眨动、头部的微小移动或纹理特征，来判断摄像头前的是真人还是照片/视频，以增强系统的安全性。
